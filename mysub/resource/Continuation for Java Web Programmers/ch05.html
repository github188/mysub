虽然我们现在明白了continuation在Java上的实现方法，但是我们当然不会想自己写一个Java的continuation实现，更不想自己写基于continuation的网络框架。那么现在在Java平台上都有些什么人在努力实现continuation？

新版的Jetty服务器用前面所说的多线程方法来部分实现continuation。另外，Apache Cocoon里有一个特别修改过的Javascript引擎支持continuation，所以在Cocoon中要用到continuation的代码一定要用Javascript而不是Java写。Jakarta commons sandbox中另有一个Javaflow Component项目，意图在Java上实现continuation。不过，目前为止Java上还没有出现一个用途广泛、使用可靠的continuation框架。
