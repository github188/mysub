前面已经我们已经对continuation有了相当的了解，也讲述了它与web之间的关系，相信您现在已经迫不及待地想试试用continuation写网络程序了吧！接下来我们就介绍Seaside框架，其网址为www.seaside.st。Seaside框架是用smalltalk语言写的，所以在学习Seaside的过程中我们也会学到不少smalltalk的知识。Seaside的核心思想是部件，这一点和tapestry很相似；但是借助于smalltalk语言和continuation，Seaside能轻易做到很多tapestry大费周章才能完成的功能。如果您喜欢tapestry，那么您很可能也会喜欢Seaside。
