2. 安装和启动Rife

Rife的安装并不复杂，基于“full stack framework”的理念，整个Rife只有一个jar文件，而且其核心部分没有任何依赖关系。不过，我们在Rife网站上可以下载到一个rife-jumpstart文件包，它让Rife开发更容易。我们只需要解压该zip文件，然后在Eclipse中选择File -> Import -> Existing Projects into Workspace，并选中rife-jumpstart的根文件夹就行了。导入完成后rife-jumpstart的结构如下图：

单击Run图标右边的小箭头，您会看到Run WebUI任务。Rife-jumpstart中自带了一个Jetty服务器以方便我们开发，这个Run WebUI任务就会启动该Jetty服务器。Jetty启动得相当快，启动成功后我们可以打开浏览器到http://localhost:8080。如果您能看到下面的页面，那么恭喜您！您可以在Rife/Eclipse的舒适环境中继续我们的continuation之旅了。
