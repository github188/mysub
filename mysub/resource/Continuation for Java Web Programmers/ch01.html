提到continuation就不能不提到continuation passing style (CPS)。两者既可以独立理解，其中任何一个又可以用来解释对方。虽然C和Java等语言不直接支持continuation，但是我们还是可以很容易的用它们写出CPS程序。所以本章中我们先介绍CPS，等到您完全理解这个概念了，我们在第二章中再正式引入continuation。

本章假设您对C有一定的了解，尤其是函数指针的使用。
