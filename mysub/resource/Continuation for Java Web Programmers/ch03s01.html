1. 安装和启动Seaside

首先您需要下载Squeak的最新版。Squeak是最受欢迎的smalltalk环境之一，支持Windows, Linux, Mac OS X和各种UNIX。Squeak当前的最新版是3.8.9。然后您需要下载Seaside的image，当前最新版是Seaside-2.5.image。Squeak是smalltalk虚拟机，就好像游戏机模拟器一样，而Seaside image呢，当然就像游戏ROM啦。Squeak不使用操作系统的窗口和文件系统，这和游戏机模拟器一样，但是熟悉Java虚拟机的读者可能对Squeak的这种运行方式不太熟悉。记住smalltalk在现代图形操作系统之前就已经出现了，因此往往把smalltalk想像成一个操作系统而不是一个应用程序会比较容易理解。双击Seaside-2.5.image，您会看到Squeak和Seaside启动：
